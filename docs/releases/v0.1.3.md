# MapleBridge Open v0.1.3

This release makes the public examples more concrete for sourcing, procurement, and supplier-matching workflows.

## What Changed

- Added category-specific buyer intents for skincare packaging and smart home electronics.
- Added supplier capability examples for packaging, skincare packaging, and smart home electronics.
- Added match explanations that show why a buyer intent and supplier capability fit.
- Added human-review notes so the examples stay clear about what should be checked before an introduction.

## Why It Matters

MapleBridge Open is not a supplier list. It is a public contract layer for representing:

- buyer intent
- supplier capability
- match explanation
- review state
- public/private workflow boundaries

The new examples make that contract easier to inspect without needing access to the production MapleBridge marketplace.
